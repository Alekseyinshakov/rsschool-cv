Code example: Maximum profit on stocks

var maxProfit = function (prices) {
maxCash = 0;
for(let i = 0; i < prices.length; i++) {
    for(let n = i; n < prices.length; n++) {
        let diff = prices[n] - prices[i]
        if(diff > maxCash) {
            maxCash = diff;
        } 
    }
}
return maxCash;
};

Description of the task

You receive an array prices, where prices[i] indicates the price of this stock on the i-th day.

You want to maximize your profits by choosing one day to buy one stock and choosing another day in the future to sell that stock.

Write a function maxProfit(prices) that returns the maximum profit you can make from this trade. If you can't make any profit, return 0.
